# Aurora Login (demo)

Scaffolded Vite + React project with Tailwind CSS and a demo `LoginPage` component.

## Quick start

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

Open the app at the URL shown by Vite (usually http://localhost:5173).

## Notes

- Social sign-ins are stubbed; replace handlers in `src/components/SocialButtons.jsx` or `src/components/LoginPage.jsx` with your real OAuth flows.
- Tailwind is configured via `tailwind.config.cjs` and `postcss.config.cjs`.
