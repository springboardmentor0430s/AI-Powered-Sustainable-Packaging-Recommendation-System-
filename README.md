# EcoPackAI

EcoPackAI is an AI-powered sustainable packaging recommendation system that predicts manufacturing cost, CO₂ environmental impact, and material safety using machine learning. The system integrates a Flask backend, Streamlit dashboard, PostgreSQL database, and BI analytics to support data-driven, eco-friendly packaging decisions.

**Features**:

- ML-based cost and CO₂ prediction

- Material safety classification

- Interactive Streamlit dashboard

- Business intelligence analytics

- PDF and Excel sustainability reports

**Tech Stack**:

- Python, Flask, Streamlit

- Random Forest, XGBoost, Scikit-learn

- PostgreSQL, Matplotlib, Altair

**Usage**:

Run the Flask backend and launch the Streamlit app to input material parameters and view predictions and sustainability insights.
